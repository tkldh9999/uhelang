# 으헤랭
엄랭을 참고해서 만든 난해한 프로그래밍 언어입니다!
제목의 유래는... 노코멘트 하겠습니다.

학교 프로젝트로 간단하게 만든겁니다.

다른 구현체를 만들 계획은 아직 없습니다.

튜링 완전을 총족합니다.

# 문법
## 연산자
‘~’은 1 증가, ‘.’은 1 감소, ‘?’은 곱셈이다.

증가, 감소, 곱셈 연산에서 혼합계산은 적용되지 않으며, ~와 .가 이어져서 하나의 항을 이룬다.

숫자들은 이 연산자들과 각종 키워드들이 더해져 하나의 항을 이루고 그 항들이 ‘?’(곱셈) 으로 연결된 형태를 지닌다.

~~~
예시)
~~~ -> 3
…. -> -4
~~~~?.. -> -8
~~~~?~~~~ -> 16
~~~

## 변수 선언
으헤 -> 0번째 변수 초기화

으헤에 -> 1번째 변수 초기화

으헤에에 -> 2번째 변수 초기화

예시)

으헤… -> 0번째 변수의 값을 -3으로 지정

으헤에~~~~ -> 1번째 변수의 값을 4로 지정

## 변수 가져오기
이야 -> 0번째 변수의 값

이야아 -> 1번째 변수의 값

이야아아 -> 2번째 변수의 값

예시)

으헤이야아 -> 0번째 변수의 값을 1번째 변수로 초기화

으헤이야.. -> 0번째 변수의 값을 (0번째 변수의 값)-2 로 초기화

## 조건문
으헿과 으헷이 있다.

으헿(값1),(값2) : (값1)이 0이면 (값2)번째 줄로 이동한다.

으헷(값1),(값2) : (값1)이 0이 ‘아니면’ (값2)번째 줄로 이동한다.

으헷,으헷 모두 (값2)가 음수일 경우 코드를 즉시 종료한다.
물론 값1에 아무것도 넣지 않으면 0으로 간주되어 조건 확인 없이 자유로운 goto가 가능해진다.

## 입력과 출력
출력 : 아쯔이, 유메 가 있다. 으헤랭에서 출력에는 줄바꿈이 없다.

아쯔이 : 숫자를 출력한다

예시) 

아쯔이~~~ -> 3 출력

아쯔이이야 -> 0번째 변수 출력



유메 : 숫자를 아스키 코드로 변환하여 출력한다

예시) 

유메 ~~~~~~~~~~?~~~~~~~~~~ -> d 출력



히요리 : 입력

예시) 

으헤히요리 -> 0번째 변수에 입력받은 숫자 대입
